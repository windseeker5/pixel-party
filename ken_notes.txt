  Option 1 - Create a script file:
  nano fix_wlan1.sh

  Copy/paste this into the file:
  #!/bin/bash
  echo "Setting up wlan1..."
  sudo ip addr add 192.168.4.1/24 dev wlan1
  sudo ip link set dev wlan1 up
  echo "Starting services..."
  sudo systemctl start hostapd
  sudo systemctl start dnsmasq
  echo "Checking wlan1..."
  ip addr show wlan1
  echo "Done! Now run: cd /home/kdresdell/Documents/DEV/PixelParty && python app.py"

  Save (Ctrl+X, Y, Enter), then run:
  chmod +x fix_wlan1.sh
  ./fix_wlan1.sh
